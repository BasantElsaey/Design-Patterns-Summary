Builder Pattern

Builder

Description

The Builder pattern is a creational design pattern that allows constructing complex objects step by step. It separates the construction of a complex object from its representation, so the same construction process can create different representations.

Advantages

  • Allows organized and readable construction of complex objects.
  • Reduces configuration errors by providing a fluent interface.
  • Provides flexibility in customizing objects without constructors with many parameters.

Use in Node.js

In Node.js, the Builder pattern is used for building HTTP requests with multiple parameters or complex data models, such as constructing query objects for databases or API payloads in libraries like Axios.

Notes

Builder is ideal when an object has many optional parameters. In Node.js, it's often used with fluent APIs for configuration objects, like in the 'request' library or custom CLI tools.

Tips

  • Return the builder instance from each method to enable method chaining.
  • Make the final object immutable by freezing it after construction.
  • Consider using it with interfaces to ensure type safety in TypeScript.

Additional Information

The Builder pattern is frequently used in Node.js for creating immutable objects, such as in the 'immutable.js' library or for building configuration objects in webpack configs.

Code Example


class House {
  walls: number;
  doors: number;
  windows: number;
  toString(): string {
    return `House with ${this.walls} walls, ${this.doors} doors, ${this.windows} windows`;
  }
}

class HouseBuilder {
  private house: House;
  constructor() { this.house = new House(); }
  setWalls(walls: number): HouseBuilder { this.house.walls = walls; return this; }
  setDoors(doors: number): HouseBuilder { this.house.doors = doors; return this; }
  setWindows(windows: number): HouseBuilder { this.house.windows = windows; return this; }
  build(): House { return this.house; }
}
          

function HouseBuilder() {
  this.house = { walls: 0, doors: 0, windows: 0 };
  return {
    setWalls: (walls) => { this.house.walls = walls; return this; },
    setDoors: (doors) => { this.house.doors = doors; return this; },
    setWindows: (windows) => { this.house.windows = windows; return this; },
    build: () => this.house
  };
}
