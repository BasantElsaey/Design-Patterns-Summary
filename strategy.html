Strategy Pattern

Strategy

Description

The Strategy pattern defines a family of algorithms, encapsulates each one, and makes them interchangeable. It allows the algorithm to vary independently from clients that use it.

Advantages

  • Allows easy swapping of algorithms without changing core code.
  • Promotes separation of concerns by isolating algorithm implementation.
  • Provides flexibility in choosing behavior at runtime.

Use in Node.js

Used for managing different payment or authentication methods, like dynamically selecting payment processors in e-commerce apps.

Notes

Strategy is great for pluggable behaviors. In Node.js, it's used in sorting algorithms for data processing or validation strategies in forms.

Tips

  • Use interfaces for strategies to ensure interchangeability.
  • Inject strategies via constructor for better testability.
  • Consider the Flyweight pattern for shared strategies to optimize memory.

Additional Information

Strategy is a key pattern in functional programming paradigms adopted in Node.js, seen in libraries like Lodash for utility functions.

Code Example


interface PaymentStrategy { pay(amount: number): string; }

class CreditCardPayment implements PaymentStrategy {
  pay(amount: number): string { return `Paid ${amount} via Credit Card`; }
}

class PayPalPayment implements PaymentStrategy {
  pay(amount: number): string { return `Paid ${amount} via PayPal`; }
}

class PaymentContext {
  private strategy: PaymentStrategy;
  constructor(strategy: PaymentStrategy) { this.strategy = strategy; }
  setStrategy(strategy: PaymentStrategy): void { this.strategy = strategy; }
  executePayment(amount: number): string { return this.strategy.pay(amount); }
}
          

function CreditCardPayment() {
  this.pay = (amount) => `Paid ${amount} via Credit Card`;
}
function PayPalPayment() {
  this.pay = (amount) => `Paid ${amount} via PayPal`;
}

function PaymentContext(strategy) {
  this.strategy = strategy;
  this.setStrategy = (newStrategy) => this.strategy = newStrategy;
  this.executePayment = (amount) => this.strategy.pay(amount);
}
