Middleware Pattern

Middleware

Description

The Middleware pattern allows processing requests in a chain of steps, where each step can modify or halt the request. It's a pipeline for handling cross-cutting concerns.

Advantages

  • Provides a flexible structure for request processing and extension.
  • Allows easy addition of features like logging or authentication without core changes.
  • Reduces coupling between application components by modularizing concerns.

Use in Node.js

Common in Express.js for managing middlewares like authentication or error handling, or in custom ZeroMQ message processing.

Notes

Middleware is essentially Chain of Responsibility applied to request pipelines. In Node.js, it's the backbone of web frameworks.

Tips

  • Pass a 'next' callback to allow chaining.
  • Handle errors in middleware to prevent breaking the chain.
  • Use async middleware for non-blocking operations in Node.js.

Additional Information

Middleware is a key pattern in web development, extended in GraphQL with Apollo Server for resolvers.

Code Example


interface Middleware { process(data: string): string; }

class LoggerMiddleware implements Middleware {
  process(data: string): string { return `Logged: ${data}`; }
}

class AuthMiddleware implements Middleware {
  process(data: string): string { return `Authenticated: ${data}`; }
}

class MiddlewareManager {
  private middlewares: Middleware[] = [];
  add(middleware: Middleware): void { this.middlewares.push(middleware); }
  execute(data: string): string {
    let result = data;
    for (const middleware of this.middlewares) {
      result = middleware.process(result);
    }
    return result;
  }
}
          

function LoggerMiddleware() {
  this.process = (data) => `Logged: ${data}`;
}
function AuthMiddleware() {
  this.process = (data) => `Authenticated: ${data}`;
}

function MiddlewareManager() {
  this.middlewares = [];
  this.add = (middleware) => this.middlewares.push(middleware);
  this.execute = (data) => {
    let result = data;
    for (const middleware of this.middlewares) {
      result = middleware.process(result);
    }
    return result;
  };
}
